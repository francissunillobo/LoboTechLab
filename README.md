# Kotlin from Scratch

Kotlin learning projects.

## EP01 Hello World

### Prerequisites

- Java 21 (or compatible JDK)
- No need to install Gradle or Kotlin globally — the Gradle wrapper handles it

### Build the project

From the `EP01 Helloworld` folder:

```powershell
cd "EP01 Helloworld"
.\gradlew.bat build
```

### Run the project

```powershell
cd "EP01 Helloworld"
.\gradlew.bat run
```

Expected output:

```
Hello, World!
```

### Clean and rebuild

```powershell
cd "EP01 Helloworld"
.\gradlew.bat clean build
```
